# Project Structure

```text
HotelManagementSystem/
├── src/
│   ├── hotel/management/system/   Java source files
│   ├── icons/                     Image resources used by the UI and PDF reports
│   └── org/jfree/chart/           Compatibility helper class
├── lib/                           Required JAR libraries
├── database/                      MySQL database import file
├── config/                        Database connection configuration
├── docs/                          User documentation
├── nbproject/                     NetBeans project configuration
├── build.xml                      Ant build file
├── manifest.mf                    Application manifest
├── README.md                      Project overview
└── .gitignore                     Ignored generated/runtime files
```

## Important Folders

### src/
Contains all Java source code and image resources.

### lib/
Contains required third-party JAR files:

- MySQL Connector/J
- iText PDF
- JFreeChart
- JCalendar
- ZXing
- rs2xml

### database/
Contains the SQL file required to create and populate the database.

### config/
Contains database connection settings.

### Runtime Folders

The following folders are generated by the application when needed and are not required in the repository:

```text
Backups/
Reports/
QR/
build/
dist/
```
